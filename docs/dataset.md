## Dataset Overview

Source: UCI Machine Learning Repository  
Description: UK based online retail transactional data  
Coverage: 01/12/2010 â€“ 09/12/2011

| Column | Description |
| --- | --- |
| InvoiceNo | 6-digit unique code assigned for each transaction. If this code starts with letter 'c', it indicates a cancellation |
| StockCode | 5-digit unique code assigned to each distinct product |
| Description | Description for each product |
| Quantity | Quantities of each product (item) per transaction |
| InvoiceDate | Day and time when each transaction was generated |
| UnitPrice | Product price per unit in sterling (GBP) |
| CustomerID | 5-digit unique code assigned to each customer |
| Country | The Country where the customer resides |
